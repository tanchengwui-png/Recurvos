
• 1. Go to your server terminal and generate SSH key:

  ssh-keygen -t ed25519 -C "tanchengwui@gmail.com"

  Press Enter for file path.
  For passphrase, press Enter twice (empty).

  2. Start SSH agent:

  eval "$(ssh-agent -s)"

  3. Add key:

  ssh-add ~/.ssh/id_ed25519

  4. Show public key:

  cat ~/.ssh/id_ed25519.pub

  5. Copy the full output (starts with ssh-ed25519 ...).
  6. Open GitHub SSH keys page:
     https://github.com/settings/keys
  7. Click New SSH key.
  8. Fill form:

  - Title: v102169 server
  - Key type: Authentication Key
  - Key: paste the copied public key

  9. Click Add SSH key.
  10. In terminal, check current remote:

  git remote -v

  11. Change remote to SSH:

  git remote set-url origin git@github.com:tanchengwui-png/Recurvos.git

  12. Confirm remote changed:

  git remote -v

  It should show git@github.com:tanchengwui-png/Recurvos.git.

  13. Test SSH connection:

  ssh -T git@github.com

  If asked continue connecting, type yes.

  14. Push:

  git push

  15. Next time, just push normally:

  git push

  No GitHub password prompt for push anymore.
